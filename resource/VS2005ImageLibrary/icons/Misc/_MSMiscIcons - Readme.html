Visual Studio Image Library: Miscellaneous Icon Files

Use Restrictions
Icon files contained in this folder are to be used in a variety of places in which .ico format is required. They may contain 16x16 and/or 32x32 image resources, and represent objects or annotate the state of objects, and must be used consistently with the descriptive name given to the file as listed below:
