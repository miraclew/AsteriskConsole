Visual Studio Image Library: 16 Color Command Art

Use Restrictions
Command icons are used to represent commands in the menu structure. These are most often action verbs, but sometimes are nouns (objects or tools) with actions associated with them. As part of a visual language, the images in this folder (or any part of the images) should be used consistent with, although not necessarily identical to, the usage described below:
